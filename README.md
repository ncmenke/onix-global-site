# ONIX LABS GLOBAL // PUBLIC INTERFACE

**CLASSIFICATION:** PUBLIC RELEASE
**SYSTEM:** MARK IV "HYPERION" TELEMETRY PORTAL
**STATUS:** ACTIVE

## MISSION
This repository contains the public-facing source code for the ONIX LABS GLOBAL digital presence. It serves as the primary visualization interface for:
1.  **Hyperion Core:** Real-time fusion stability metrics.
2.  **Sentinel Architecture:** Photonic lattice throughput monitoring.
3.  **The Kernel:** Symplectic physics engine status.

## ARCHITECTURE
* **Frontend:** React 18 + TypeScript (Vite)
* **Styling:** TailwindCSS + Framer Motion
* **Hosting:** Sovereign Cloud / Edge Network
* **Security:** Client-side only. No direct bridge to the physics kernel.
